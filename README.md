# BreatheWith App Support Site

Static help, FAQ, and privacy policy pages for **BreatheWith** (iOS and Android). The live site is published with GitHub Pages.

**Support site:** [https://d108.github.io/breathewith-app-support/](https://d108.github.io/breathewith-app-support/)

Source files in this repository (`index.html`, `privacy-policy.html`, shared assets) are deployed to the `gh-pages` branch by [`.github/workflows/pages.yml`](.github/workflows/pages.yml) on pushes to `main`.
